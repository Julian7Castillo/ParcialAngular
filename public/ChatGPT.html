Para agregar una imagen en tu estilo CSS, puedes hacerlo utilizando varias propiedades CSS, dependiendo de cómo quieras usar la imagen. A continuación, te doy las formas más comunes para hacerlo:

1. Usar una imagen como fondo (background-image)

Puedes utilizar la propiedad background-image para establecer una imagen de fondo para un elemento. Este es un método común para usar imágenes en elementos como div, header, section, etc.

Ejemplo:

css
/* Estilo para agregar una imagen de fondo a un contenedor */
.container {
  width: 100%;
  height: 300px;
  background-image: url('ruta/a/tu-imagen.jpg');
  background-size: cover; /* La imagen cubre todo el contenedor */
  background-position: center; /* Centrar la imagen */
  background-repeat: no-repeat; /* No repetir la imagen */
}
  • background-size: cover; hace que la imagen cubra todo el área del elemento, ajustándose según sea necesario.
  • background-position: center; centra la imagen en el elemento.
  • background-repeat: no-repeat; asegura que la imagen no se repita.

Ejemplo con una URL externa:

css
.container {
  background-image: url('https://example.com/imagen.jpg');
}

2. Usar una imagen en un pseudo-elemento (::before o ::after)

También puedes agregar una imagen usando pseudo-elementos como ::before o ::after. Este enfoque es útil si quieres agregar una imagen decorativa sin necesidad de ponerla en el HTML directamente.

Ejemplo:

css
/* Usando ::before para agregar una imagen antes de un elemento */
.image-container::before {
  content: '';
  display: block;
  width: 100px;
  height: 100px;
  background-image: url('ruta/a/tu-imagen.png');
  background-size: cover;
  background-position: center;
}

En este ejemplo, estamos usando el pseudo-elemento ::before para agregar una imagen antes del contenido de .image-container.

3. Establecer una imagen como una propiedad (list-style-image o border-image)

Algunas propiedades específicas permiten establecer imágenes directamente, como list-style-image para listas, o border-image para bordes.

Ejemplo con list-style-image:

css
ul {
  list-style-image: url('ruta/a/tu-icono.png');
}

Ejemplo con border-image:

css
.box {
  border: 10px solid transparent;
  border-image: url('ruta/a/tu-imagen.png') 30 round;
}

4. Usar imágenes con HTML y aplicando estilos CSS

Si prefieres insertar la imagen directamente en el HTML usando la etiqueta <img>, puedes seguir aplicando estilos desde el CSS.

HTML:

html
<img src="ruta/a/tu-imagen.jpg" class="mi-imagen">

CSS:

css
.mi-imagen {
  width: 300px;
  height: auto;
  border-radius: 10px; /* Bordes redondeados */
}

Resumen:

  • background-image: Para imágenes de fondo de un elemento.
  • ::before o ::after: Para insertar imágenes de forma decorativa sin afectar el HTML.
  • list-style-image o border-image: Para imágenes en listas o bordes.
  • Estilos aplicados a <img>: Para controlar el tamaño, bordes, o cualquier estilo de una imagen insertada en el HTML.

Recuerda que la ruta de la imagen puede ser relativa (dependiendo de la ubicación del archivo CSS) o una URL completa si es una imagen externa.

2. Uso de imágenes en CSS dentro de Angular

Si tienes una imagen ubicada en la carpeta assets, la ruta en tu CSS debe ser relativa a la carpeta src porque Angular compila y organiza los archivos de manera que el contenido en assets es accesible de forma pública en el proyecto.

Ejemplo:

Supongamos que tienes la siguiente estructura de archivos:

css
src/
  assets/
    images/
      logo.png
  styles/
    styles.css

Y quieres usar la imagen logo.png en tu archivo styles.css. Debes referenciar la imagen así:

css
/* Ruta relativa a la carpeta de assets */
.logo {
  background-image: url('/assets/images/logo.png');

3. Uso de imágenes en HTML dentro de Angular

Si estás insertando imágenes directamente en el HTML de tus componentes Angular, como en un archivo .html de un componente, también puedes usar la ruta relativa desde la carpeta assets.

Ejemplo en HTML:

html
<img src="assets/images/logo.png" alt="Logo">

4. Diferencias entre rutas relativas y absolutas en Angular

  • Rutas relativas: En el CSS o HTML, puedes usar rutas relativas si las imágenes están ubicadas en el mismo nivel que el archivo donde las usas. Pero para simplificar, es mejor seguir la convención de usar la carpeta assets para todos los archivos estáticos.

  • Rutas absolutas: Como la aplicación Angular normalmente se sirve desde la raíz (/), puedes usar rutas como /assets/... para acceder a cualquier archivo dentro de la carpeta assets.

5. Uso de rutas en archivos SCSS (preprocesador CSS) en Angular

Si estás utilizando SCSS en lugar de CSS, el comportamiento es el mismo. Las rutas de las imágenes siguen siendo relativas a la carpeta assets.

scss
/* Ejemplo usando SCSS */
.banner {
  background-image: url('/assets/images/banner.jpg');
  background-size: cover;
  background-position: center;

Durante la compilación, Angular copia automáticamente todo el contenido de la carpeta src/assets al directorio de distribución (dist/). Esto asegura que las imágenes y otros recursos estén disponibles en la ruta /assets/ cuando la aplicación se despliega en producción.

Resumen:

  • Coloca tus imágenes en la carpeta src/assets/.
  • En CSS o SCSS, usa la ruta /assets/ para referenciar imágenes.
  • En los archivos HTML de los componentes Angular, también puedes referenciar las imágenes con la ruta assets/... (sin la barra inicial).
  • Angular copiará automáticamente todo lo que esté en src/assets a la carpeta de distribución, lo que hace que las imágenes estén disponibles en /assets/ cuando se despliegue la aplicación.

css
src/
  assets/

.banner {

    arduino
    src/

    En tu archivo angular.json, busca la sección "assets" dentro de "build" y agrega la carpeta public. Se verá algo así:

    json
    {
  "projects": {
    "your-app-name": {
      "architect": {
        "build": {

Recomendación

Aunque puedes usar una carpeta personalizada como public, se recomienda seguir la convención de Angular y colocar tus recursos estáticos dentro de src/assets, ya que Angular está optimizado para trabajar con esta estructura.

Usando la carpeta assets evitarás configuraciones adicionales y posibles problemas al desplegar tu aplicación.

Para centrar un texto verticalmente en CSS, hay varias formas dependiendo del contexto y de qué tipo de elemento estés utilizando. Aquí te doy varias soluciones, dependiendo de las propiedades CSS que puedas necesitar.

1. Centrar con Flexbox

El uso de Flexbox es uno de los métodos más fáciles y versátiles para centrar tanto horizontal como verticalmente. Esto funciona tanto con elementos de bloque (div, section, etc.) como con elementos de texto.

Ejemplo:

css
.container {
  display: flex;
  justify-content: center;  /* Centra horizontalmente */
  align-items: center;      /* Centra verticalmente */
  height: 300px;            /* Asegúrate de que el contenedor tenga altura */
  border: 1px solid black;  /* Solo para que sea visible el ejemplo */
}

HTML:

html
<div class="container">
  <p>Texto centrado vertical y horizontalmente</p>
</div>
  • justify-content: center; centra el contenido horizontalmente.
  • align-items: center; centra el contenido verticalmente.
  • height es necesario porque el contenedor debe tener una altura para poder centrar el texto verticalmente.

2. Centrar con Grid

Si prefieres usar CSS Grid, también puedes centrar fácilmente el texto verticalmente de manera similar a Flexbox.

Ejemplo:

css
.container {
  display: grid;
  place-items: center;  /* Centra tanto horizontal como verticalmente */
  height: 300px;
  border: 1px solid black;
}

HTML:

html
<div class="container">
  <p>Texto centrado</p>
</div>
  • place-items: center; es una forma abreviada para centrar tanto en el eje horizontal como en el vertical en un contenedor con Grid.

3. Centrar usando propiedades line-height (para una línea de texto)

Si el texto es de una sola línea y quieres centrarlo dentro de un contenedor con altura fija, puedes usar line-height igual a la altura del contenedor.

Ejemplo:

css
.container {
  height: 300px;
  line-height: 300px;  /* Hace que el texto se alinee verticalmente */
  text-align: center;  /* Centra el texto horizontalmente */
  border: 1px solid black;
}

HTML:

html

  Texto centrado

  • Nota: Este método solo funciona bien para texto de una sola línea. Si tienes múltiples líneas, el texto se desajustará.

4. Centrar con position y transformaciones (translateY)

Este método utiliza las propiedades position y transform para centrar el texto verticalmente.

Ejemplo:

css
.container {
  position: relative;
  height: 300px;
  border: 1px solid black;
}

.text {
  position: absolute;
  top: 50%;           /* Mueve el texto al 50% desde la parte superior */
  left: 50%;          /* (Opcional) Para centrar también horizontalmente */
  transform: translate(-50%, -50%);  /* Ajusta el texto a la mitad de su propia altura y anchura */
}

HTML:

html
<div class="container">
  <p class="text">Texto centrado</p>
</div>
  • position: absolute; saca el texto del flujo normal del documento y lo coloca en relación al contenedor padre.
  • top: 50%; mueve el texto hacia la mitad de la altura del contenedor.
  • transform: translate(-50%, -50%); ajusta el texto hacia atrás para que quede perfectamente centrado.

5. Centrar usando table y vertical-align (obsoleto para layout, pero útil en algunos casos)

Este método usa propiedades de tablas y vertical-align. Aunque es una técnica más antigua, sigue siendo útil en algunos casos.

Ejemplo:

css
.container {
  display: table;
  height: 300px;
  width: 100%;
  border: 1px solid black;
}

.text {
  display: table-cell;
  vertical-align: middle;  /* Centra verticalmente */
  text-align: center;      /* Centra horizontalmente */
}

HTML:

html
<div class="container">
  <p class="text">Texto centrado</p>
</div>

Resumen de técnicas:

  • Flexbox: Ideal para layouts flexibles y modernos. display: flex; con align-items: center;.
  • Grid: Similar a Flexbox pero más potente en algunos casos. display: grid; con place-items: center;.
  • line-height: Útil para centrar una sola línea de texto.
  • position + transform: Buena opción para layouts precisos con transformaciones.
  • table + vertical-align: Método antiguo, pero aún útil en casos limitados.

De estas opciones, Flexbox y Grid son las soluciones más modernas y recomendadas para centrar texto en CSS.

En CSS, para cambiar el color de un elemento cuando está seleccionado, puedes utilizar el pseudoelemento ::selection. Este pseudoelemento se aplica a la parte del contenido que ha sido resaltada o seleccionada por el usuario, generalmente con el cursor o al usar el teclado.

Ejemplo básico:

css
::selection {
  background-color: #ffcc00; /* Color de fondo cuando el texto está seleccionado */
  color: #000000; /* Color del texto seleccionado */
}
  • background-color: Cambia el color de fondo cuando el texto está seleccionado.
  • color: Cambia el color del texto que está seleccionado.

Ejemplo completo:

html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo ::selection</title>
  <style>
    ::selection {
      background-color: #4CAF50; /* Verde */
      color: white; /* Texto en blanco */
    }

    /* Se pueden especificar también estilos solo para elementos específicos */
    p::selection {
      background-color: #FF6347; /* Tomate */
      color: white; /* Texto en blanco */
    }
  </style>
</head>
<body>
  <h1>Selecciona este texto</h1>
  <p>Este es un párrafo con un estilo de selección diferente. Selecciónalo y verás el cambio de color.</p>
</body>
</html>

Detalles importantes:

  1. El pseudoelemento ::selection solo afecta al contenido dentro del elemento que el usuario selecciona (como texto).
  2. No todas las propiedades CSS funcionan con ::selection. Solo las siguientes propiedades están permitidas:
    • color
    • background-color
    • text-shadow
    • cursor

Con esto, puedes controlar el estilo visual del texto seleccionado para mejorar la experiencia de usuario o ajustarlo al diseño de tu sitio.
